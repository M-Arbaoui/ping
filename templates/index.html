API DOCUMENTATION

POST /api/ping

إجراء اختبار ping وإرجاع النتائج بصيغة JSON.

curl -X POST http://localhost:5000/api/ping \
  -H "Content-Type: application/json" \
  -d '{"hostname": "google.com"}'

POST /api/validate

التحقق من صحة عنوان IP أو اسم نطاق.

curl -X POST http://localhost:5000/api/validate \
  -H "Content-Type: application/json" \
  -d '{"hostname": "google.com"}'
